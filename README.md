# 📝 Swift Task Manager

A simple command-line Task Manager built using Swift.

## 🚀 Features

* Add tasks
* View tasks
* Delete tasks
* Interactive CLI menu

## 🛠️ Tech Used

* Swift
* Foundation Library

## ▶️ How to Run

### On Windows (using Swift CLI)

```bash
swift main.swift
```

### Online

Use platforms like Replit to run the project.

## 📸 Demo

```
===== Task Manager =====
1. Add Task
2. View Tasks
3. Delete Task
4. Exit
```
